# CRM Demo Project

A desktop CRM application built with Electron, React, TypeScript, and Vite.

## Features

- Multi-step Order Creation
- Exportable reports and tables
- Filtering by date range, location, payment type, and service type
- Real-time data filtering and searching
- Excel export functionality with progress tracking
- Responsive data tables with custom UI components

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Electron
- Electron Builder

## Getting Started

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Package the Electron app:

```bash
npm run build
```

## Key Skills Demonstrated

- Desktop application development with Electron
- Complex state management and data filtering
- TypeScript and modern React patterns
- Component-driven architecture
- File export functionality
- Responsive UI design
